# ProjetoAgroControl

Projeto Gestão de Configuração



\# Requisitos do Sistema AgroControl



\## Funcionalidades

\- Cadastro de propriedades rurais

\- Controle de plantio e colheita

\- Registro de aplicação de insumos

\- Relatório de produtividade por safra



\## Objetivo

Auxiliar produtores rurais no controle técnico da produção agrícola.
